# Приложение для поиска книг

### Установка и запуск
Склонировать репозиторий
```
    git clone https://github.com/TheHarald/books-frontend
```
Войти в папку с проектом
```
cd ./books-frontend
```
Установить зависимости
```
npm i
```
Запустить 
```
npm run start
```

### Запуск в докер контейнере
```
docker-compose up -d
```
